## About This App

Accept Connect is a cross-platform mobile application that enables secure message exchange between users through multiple interaction methods:

- **Link-based exchange**: Share secure links for message requests
- **Push notifications**: In-app notifications for installed users
- **Proximity-based exchange**: Device-to-device message delivery using Bluetooth/BLE

The app supports authentication via username/password, Google, Facebook, and national digital identity systems (SPID, CIE).

Built with React Native + Expo 54 for iOS, Android, and Web platforms.
